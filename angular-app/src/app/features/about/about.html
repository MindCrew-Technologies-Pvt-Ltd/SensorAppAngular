About This Application

This Angular application demonstrates modern web development practices and architecture patterns.

Application Structure

  • Core: Services, guards, and interceptors that are used application-wide
  • Shared: Reusable components, directives, and pipes
  • Features: Lazy-loaded feature modules for better performance

Technology Stack

  • Angular (Latest Stable Version)
  • TypeScript with Strict Mode
  • Standalone Components
  • Angular Router for Navigation
